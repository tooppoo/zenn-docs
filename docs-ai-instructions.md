# AI指示: mdファイル作成後のLint運用ルール

このリポジトリでAIエージェントがMarkdownファイルを新規作成・更新した場合、以下を必須とする。

## 必須ルール

1. mdファイル作成・更新後、必ずLintを実行する。
2. Lintが失敗した場合、失敗内容を修正して再実行する。
3. Lintが成功するまで、修正と再実行を繰り返す。

## 実行コマンド

```bash
pnpm test
```

## 適用対象

- `*.md` の新規作成
- `*.md` の既存編集
- PR作成前の最終確認

## 目的

- 表記ゆれ・文体混在・助詞重複などの品質劣化を防ぐ。
- AI生成文の見落としを機械的に検出する。
- 最低限の文章品質を担保した状態でPRを作成する。
